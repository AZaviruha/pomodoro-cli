# pomodoro-cli
haskell newbie project

```shell
> pomodoro-cli -h

Help Options:                   
  -h, --help                    
    Show option summary.        
  --help-all                    
    Show all help options.      
                                
Application Options:            
  --alarm :: text               
    Path to alarm sound file    
    default: "./audio/alarm.wav"
  --pomodoro :: int64           
    Pomodoro length             
    default: 25                 
  --shortBreak :: int64         
    Short break length          
    default: 5                  
  --longBreak :: int64          
    Long break length           
    default: 20
```
